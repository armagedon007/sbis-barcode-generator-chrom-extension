# СБИС Barcode Generator - Chrome Extension

Расширение для автоматической генерации штрихкодов EAN-13 в системе СБИС.

## Установка

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode)
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку с расширением

## Использование

1. Откройте страницу СБИС (online.sbis.ru или ret.sbis.ru)
2. Найдите поле ввода штрихкода с кнопкой генерации (иконка молнии)
3. Убедитесь что выбран тип "Штрихкод"
4. Нажмите на кнопку с иконкой молнии
5. Расширение автоматически сгенерирует уникальный штрихкод EAN-13

## Настройки

- **Префикс**: начало штрихкода (по умолчанию: 62)
- **Начальный номер**: с какого номера начинать (по умолчанию: 100000)

## Обновление

Расширение поддерживает автообновление из GitHub:

1. Нажмите на иконку расширения в панели инструментов Chrome
2. Нажмите кнопку "Проверить обновления"
3. Если доступна новая версия, нажмите "Загрузить обновление"
4. Перезагрузите расширение в `chrome://extensions/`

### Настройка автообновления

Для работы автообновления нужно:

1. Создать репозиторий на GitHub (например: `username/sbis-barcode-generator`)
2. Загрузить файлы расширения в репозиторий
3. Отредактировать `updater.js` и заменить `YOUR_USERNAME/sbis-barcode-generator` на ваш репозиторий
4. При выпуске новой версии обновите `version` в `manifest.json` и загрузите изменения в GitHub
