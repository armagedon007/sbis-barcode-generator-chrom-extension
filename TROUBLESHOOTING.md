# Устранение проблем

## Расширение не работает в Яндекс браузере

### Шаг 1: Проверка установки расширения

1. Откройте `browser://extensions/` (или `chrome://extensions/`)
2. Найдите "СБИС Barcode Generator"
3. Убедитесь что:
   - Расширение включено (переключатель активен)
   - Нет ошибок загрузки
   - Версия: 1.0.2 или выше

### Шаг 2: Проверка разрешений

1. В списке расширений нажмите "Подробнее"
2. Проверьте что есть доступ к сайтам:
   - `https://online.sbis.ru/*`
   - `https://ret.sbis.ru/*`
   - `https://*.sbis.ru/*`

### Шаг 3: Проверка консоли background

1. В `browser://extensions/` найдите расширение
2. Нажмите "Проверить представления: service worker" (или "background page")
3. Откроется консоль разработчика
4. Должны быть сообщения:
   ```
   [SBIS Barcode] Background service worker запущен
   [SBIS Barcode] Service worker активен, версия: 1.0.2
   ```

Если сообщений нет - расширение не загрузилось. Попробуйте:
- Перезагрузить расширение (кнопка обновления)
- Удалить и установить заново

### Шаг 4: Проверка консоли страницы

1. Откройте СБИС (online.sbis.ru или ret.sbis.ru)
2. Нажмите F12 (открыть консоль разработчика)
3. Обновите страницу (F5)
4. В консоли должны появиться сообщения:
   ```
   [SBIS Barcode] Скрипт content.js загружен!
   [SBIS Barcode] document.readyState: ...
   [SBIS Barcode] Расширение инициализировано
   [SBIS Barcode] URL: https://...
   [SBIS Barcode] Обработчик клика установлен
   ```

Если сообщений нет:
- Проверьте что вы на правильном сайте (sbis.ru)
- Проверьте разрешения расширения
- Попробуйте перезагрузить страницу

### Шаг 5: Проверка работы при клике

1. Откройте страницу создания/редактирования номенклатуры
2. Найдите поле "Код" с кнопкой молнии
3. Выберите тип "Штрихкод"
4. Откройте консоль (F12)
5. Нажмите на кнопку молнии
6. В консоли должны появиться сообщения:
   ```
   [SBIS Barcode] Клик обнаружен: ...
   [SBIS Barcode] Кнопка найдена: ...
   [SBIS Barcode] Иконка молнии найдена: ...
   [SBIS Barcode] Контейнер найден: ...
   [SBIS Barcode] Тип кода: Штрихкод
   [SBIS Barcode] Это штрихкод? true
   [SBIS Barcode] Поле ввода найдено: ...
   [SBIS Barcode] Генерация штрихкода...
   [SBIS Barcode] Сгенерирован код: ...
   [SBIS Barcode] Код вставлен в поле
   ```

## Частые проблемы

### Расширение не видит кнопку молнии

Возможно структура страницы изменилась. Скопируйте HTML код кнопки:
1. Откройте инструменты разработчика (F12)
2. Выберите инструмент "Выбрать элемент" (стрелка в углу)
3. Наведите на кнопку молнии
4. Скопируйте HTML код и отправьте разработчику

### Код не вставляется в поле

Проверьте в консоли:
- Есть ли сообщение "Поле ввода найдено"?
- Есть ли ошибки после "Генерация штрихкода"?

### Ошибка проверки уникальности

Проверьте:
- Вы авторизованы в СБИС?
- Есть ли доступ к API?
- Проверьте консоль на наличие ошибок сети

## Совместимость

Расширение протестировано на:
- Google Chrome 120+
- Microsoft Edge 120+
- Яндекс Браузер (на базе Chromium)

Manifest v3 требует:
- Chrome 88+
- Edge 88+
- Яндекс Браузер 21.2+

## Получение помощи

Если проблема не решена:
1. Соберите информацию из консоли (скриншоты)
2. Укажите версию браузера
3. Укажите версию расширения
4. Опишите шаги воспроизведения
5. Создайте issue на GitHub: https://github.com/armagedon007/sbis-barcode-generator-chrom-extension/issues
