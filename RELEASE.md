# Инструкция по выпуску новой версии

## Текущая версия: 1.0.0

## Как выпустить обновление

### 1. Внесите изменения в код

Отредактируйте нужные файлы (content.js, popup.js и т.д.)

### 2. Обновите версию в manifest.json

```json
{
  "version": "1.0.1"
}
```

Версионирование:
- 1.0.0 → 1.0.1 - исправление багов
- 1.0.0 → 1.1.0 - новые функции
- 1.0.0 → 2.0.0 - крупные изменения

### 3. Закоммитьте и отправьте в GitHub

```bash
cd chrome
git add .
git commit -m "Release v1.0.1: описание изменений"
git push
```

### 4. Пользователи получат обновление

Пользователи смогут:
1. Нажать на иконку расширения
2. Нажать "Проверить обновления"
3. Увидеть новую версию и загрузить её
4. Перезагрузить расширение

## Пример коммита

```bash
git add .
git commit -m "Release v1.0.1: Исправлена ошибка генерации кода при пустом префиксе"
git push
```

## Проверка обновления

После push проверьте что файлы доступны:
```
https://raw.githubusercontent.com/armagedon007/sbis-barcode-generator-chrom-extension/main/manifest.json
```
