# Установка расширения СБИС Barcode Generator

## Быстрая установка

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку `chrome` из этого проекта
5. Расширение установлено! Иконка появится в панели инструментов

## Первая настройка

1. Нажмите на иконку расширения в панели инструментов
2. Проверьте настройки:
   - Префикс: 62 (по умолчанию)
   - Начальный номер: 100000 (по умолчанию)
3. При необходимости измените и нажмите "Сохранить"

## Использование

1. Откройте СБИС (online.sbis.ru или ret.sbis.ru)
2. Перейдите в раздел создания/редактирования номенклатуры
3. Найдите поле "Код" с выпадающим списком типа кода
4. Выберите тип "Штрихкод"
5. Нажмите на кнопку с иконкой молнии (⚡)
6. Расширение автоматически сгенерирует уникальный штрихкод EAN-13

## Как это работает

- Расширение генерирует штрихкоды последовательно: 62100000, 62100001, 62100002...
- Каждый код проверяется на уникальность через API СБИС
- Если код уже существует, генерируется следующий
- Последний использованный номер сохраняется в настройках расширения

## Обновление расширения

1. Нажмите на иконку расширения
2. Нажмите "Проверить обновления"
3. Если доступна новая версия - загрузите её
4. Перезагрузите расширение в `chrome://extensions/`

## Устранение проблем

### Расширение не работает
- Проверьте что вы на странице СБИС (online.sbis.ru или ret.sbis.ru)
- Убедитесь что выбран тип "Штрихкод"
- Откройте консоль разработчика (F12) и проверьте ошибки

### Не генерируется код
- Проверьте настройки расширения
- Убедитесь что префикс содержит только цифры
- Проверьте что начальный номер больше 0

### Ошибка проверки уникальности
- Проверьте что вы авторизованы в СБИС
- Убедитесь что у вас есть доступ к API

## Удаление расширения

1. Перейдите в `chrome://extensions/`
2. Найдите "СБИС Barcode Generator"
3. Нажмите "Удалить"
